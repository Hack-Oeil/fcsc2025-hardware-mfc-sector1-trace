# FCSC 2025 Grand classic hotel

Pendant un voyage d’affaires vous résidez au Grand Classic Hotel, sur la côte de Mifare. À votre retour de votre footing matinal, vous réalisez que vous avez perdu votre badge d’accès à votre chambre.

Les employés de la réception ne peuvent pas vous aider car le technicien est en vacances. Mais ils trouvent une trace de debug sur son PC, qu’il a faite en enregistrant la communication entre le badge et le verrou avec un Proxmark pendant un exercice de sécurité physique.

Ils acceptent de vous laisser l’examiner pour y trouver le flag.

La technologie est Mifare Classic, qui est basée sur ISO 14443.
